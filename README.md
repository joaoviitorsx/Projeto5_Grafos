# T5 - Coloracao de Grafos com DSatur

Projeto em Java para modelar o mapa politico do Brasil como grafo nao direcionado e aplicar o algoritmo DSatur.

## Estrutura

```text
T5/
├── README.md
├── dados/
│   └── brasil.txt
└── src/
    ├── Main.java
    ├── Graph.java
    └── GraphColoringDSatur.java
```

## Compilacao

```bash
javac src/*.java
```

## Execucao

```bash
java -cp src Main dados/brasil.txt
```
